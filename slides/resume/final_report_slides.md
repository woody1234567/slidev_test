# 疫情前後趨勢分析

FunNow 訂單表現亮眼

- 📉 **整體餐飲業**：因為三級警戒，疫情間營業額明顯下滑
- 🚀 **FunNow**：訂單數卻不降反升！
  - 主要動力來自於 **外帶產品**
  - 解封後客單價成長率陡峭
- 🍽️ **全新數位趨勢**
  - **獨食轉型**：過去多人分食轉為一人一份
  - **體驗延續**：以 QRcode 連結播放清單/主廚講解
  - **話題延續 x 獨家限定**：與外送車隊合作一次送到家

# 前五大具有開發潛力的地區

經過指標篩選，最終鎖定以下五個具有開發潛力的地區：

<v-clicks>

1. 🥇 **新北市 永和區** 
   - 人口密度高，雖偏向住宅區且過去非重點開發，但模型顯示具有**極高潛力**。
2. 🥈 **台北市 中山區** 
   - 消費者熱愛用餐地，餐點類別豐富且近商圈，各項指標皆獲高分。
3. 🥉 **新北市 三重區** 
   - 著名三和夜市與住宅區，小餐廳滿足日常需求，訂單相關面向表現優異。
4. **台北市 士林區**
5. **台北市 萬華區**

</v-clicks>

## 下一步 (BD Team 可延伸方向)

1. **質化訪談 (Focus Group)**
   - 了解目前餐飲使用者對各品牌的使用印象、喜歡的餐廳類型。
   - 詢問過去停止使用 FunNow 的原因。
2. **量化研究 (問卷)**
   - 透過完整市場調查，了解真實想法，優化 BD 開發方向與行銷規劃。
3. **優化模型**
   - 納入更多無提供的數據 (如餐廳名稱、GMV等) 使樣本數更充足。
   - 綜合考量過去 BD 篩選準則，結合質化指標放進最終版準則。
